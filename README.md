# chrome-tab-recorder

Google Chrome Extensions Tab Recorder

## Install

- [Chrome Web Store](https://chromewebstore.google.com/detail/instant-tab-recorder/giebbnikpnedbdojlghnnegpfbgdecmi)

## Development

### Build

```sh
npm install
npm build
```

```text

```
